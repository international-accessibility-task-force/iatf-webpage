IATF Developers

The IAFT Developers are responsible for creating open-source software that covers completely a request filled by an IATF user. The source code must live under the GPL License. This way, all of the community can benefit from it.

We will be thrilled if you join our discord as an IATF developer and helps IATF to make technologies accessible for everyone.

You can send accessibility requests for this webpage to admin@internationalaccessibilitytaskforce.com
